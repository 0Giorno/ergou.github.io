# 滇池的冬日

冬日傍晚的滇池，水面平静如镜，远处的西山在暮色中渐渐模糊。

![滇池暮色](/images/sample/yunnan.jpg "滇池|冬日暮色")

风很轻，水面上偶尔有几只红嘴鸥掠过，划出一道道细纹。夕阳把天边染成橙红色，倒映在水中，像是两条平行的光带，一条在天上，一条在水里。

# 等待日落

到滇池看日落的人不算多，三三两两散布在岸边。有人架着脚架，有人举着手机，也有人什么都不做，就静静站着看。

{{< poem >}}
落日是温柔的
它不刺眼
只是慢慢沉下去
把所有的颜色
都留给了天空和水面
{{< /poem >}}

日落后天色暗得很快，红嘴鸥也归巢了，只剩水波轻轻拍岸的声音。
